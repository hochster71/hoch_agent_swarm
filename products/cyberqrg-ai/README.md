# CyberQRG-AI

Offline-first QR code security scanner and validator.

---

## 1. Local Scaffolding
* This is a local-only prototype constructed during HAS/HASF R2 Sprint 2.
* Zero external internet dependency.

---

## 2. Running Locally
1. Install dependencies:
   `npm install`
2. Run test suites:
   `npm run test`
3. Build the TypeScript package:
   `npm run build`
