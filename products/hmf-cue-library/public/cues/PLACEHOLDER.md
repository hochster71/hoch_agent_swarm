# PLACEHOLDER — no real cue audio yet

This folder is where license-cleared instrumental cue packs will live once HMF produces and
clears them. **There is intentionally NO audio here.** Shipping a placeholder tone as if it were
a real product would be FAKE GREEN.

Per the spec (`docs/factories/products/HMF_cue_library.md`), the first real pack must be:

- 3 original instrumental audio files (loops/stings/beds), non-empty and real audio;
- a `LICENSE.txt` with an explicit usage license per cue;
- a `manifest.json` listing each cue + duration;
- passing the license gate + an automated check confirming no vocal stem and no named-artist metadata.

Until that validated, license-clean pack exists, this product is a landing page + checkout scaffold
only, not a deliverable library.
