# Golden Dataset Evals Directory

This directory hosts the golden test cases and LLM evaluation templates to verify agent output safety and quality.

## Golden Cases Structure
* `evals/golden/hasf_scoring_agent/`: Seed cases validating scoring agent outputs.
* `evals/golden/evidence_agent/`: Seed cases validating evidence generator outputs.
* `evals/judges/`: Configured prompts for G-EVAL criteria grading.
