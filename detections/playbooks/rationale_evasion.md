# Playbook: Rationale Evasion Evasion Attempt
## Trigger
SIEM detects rationale quality check rejections or repetitive rationales.
## Severity
Medium.
## Immediate Actions
1. Audit executing task and parameters.
2. Confirm if rationale provided by user contains bypass attempts.
3. Pause task if malicious behavior is suspected.
## Recovery
Re-run with manual operator input override.
