# HOCH Private-First Doctrine

* **Status**: MANDATORY / repo-wide enforcement.
* **Core Rule**: Build private. Prove private. Monetize apps. Protect the brain.

---

## 1. Private Proof First
* Under no circumstances should prompt registries, red-team playbooks, scoring logic, evidence ledgers, or orchestration logic be disclosed publicly.
* All external engagement with companies or investors remains strictly frozen.
* Public platform reveals or platform deployment announcements are held.

---

## 2. App-Store Exception Path
* Public monetization is permitted ONLY for compiled, finished standalone apps. The private swarm architecture, registry files, and pipeline engines must not be packaged or exposed in these apps.
* Remote relay private runtimes may continue executing silently.
