## 8. Demo Operator Quickstart
1. **Start Backend**: Run `uv run python backend/main.py` in directory root.
2. **Confirm LM Studio**: Make sure port `1234` returns local models.
3. **Confirm Ollama**: Make sure port `11434` returns local tags.
4. **Open Command Center**: Navigate to [http://127.0.0.1:8000/prototype/prompt-brain](http://127.0.0.1:8000/prototype/prompt-brain).
5. **Open RC1 Demo Mode**: Confirm the banner shows **RC1 Pilot Mode**.
6. **Run Demo Workflow**: Run a sample NIST SP 800-53 control analysis.
7. **Show Evidence Ledger**: Display dynamic QA scoring records.
8. **Show Red-Team Output**: Verify gap alerts and timeframes.
9. **Show Paid Pilot Offer**: Present the Starter or GovCon Team pilot price tier.
10. **Capture Feedback**: Use CLI or UI to log interest or objections.
