# HOCH Prompt Brain — Security and Data Boundary

## 1. Workstation & Container Execution
* Prompt Brain runs entirely locally using containerized model adapters (LM Studio or Ollama). Zero external network calls are performed.
* **Sensitive Data Block**: Under no circumstances should controlled (CUI), classified, export-controlled, or proprietary program evidence be uploaded in Demo Mode.
* Private network deployments must be configured and validated before processing actual program evidence.
