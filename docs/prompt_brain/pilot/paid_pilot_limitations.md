# HOCH Prompt Brain — Paid Pilot Limitations

Reviewers must be fully aware of the following scope limits:
* **No Regulatory Approval**: Prompt Brain does NOT have regulatory authorization authority. Authorizing Officials (AO) and human assessors retain all final approval authority.
* **Air-gapped Workstations Only**: Local model performance is bound by workstation CPU/GPU limits.
* **No Compliance Guarantees**: Prompt Brain identifies omission gaps based on prompt heuristics; it does not certify total compliance.
