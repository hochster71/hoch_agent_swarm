# HOCH Prompt Brain — Pricing Tiers

## 1. Free External Review
* **Price**: $0
* **Duration**: 15–30 minutes (one-time).
* **Included**: Tab 10 Command Center access running on sanitized demo dataset.
* **Excluded**: Custom workflow configuration, local workstation installation support.
* **Support Level**: Self-service documentation.
* **Conversion Path**: Request trial key for Starter Pilot.

## 2. Starter Pilot
* **Price**: $499–$999/month.
* **Included**: 3 active compliance workflows, 2 user seats.
* **Excluded**: Custom STIG template mapping, on-prem support.
* **Support Level**: Email support (24h SLA).
* **Conversion Path**: Request upgrade to GovCon Team Pilot.

---

## 3. GovCon Team Pilot
* **Price**: $2,500–$5,000/month.
* **Included**: 5 active compliance workflows, up to 15 user seats, local workstation container deployment support.
* **Excluded**: Custom software modifications.
* **Support Level**: Priority Slack & Teams support (4h SLA).
* **Conversion Path**: Annual enterprise license.
