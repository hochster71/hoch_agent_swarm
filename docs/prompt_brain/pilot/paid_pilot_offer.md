# HOCH Prompt Brain — Paid Pilot Offer Package

This document proposes a structured paid pilot program for defense contractors (GovCons) and compliance teams.

---

## 1. Pilot Terms
* **Duration**: 14 to 30 Days.
* **Pricing**: Flat team fee (details in pricing schedule).
* **Onboarding support**: Full assistance from HOCH engineers to configure local model environments (LM Studio / Ollama workstation containers).

---

## 2. Safety Guidelines
* All local instances run air-gapped without outbound network requests.
* Pilot uses the **Sanitized Demo Dataset** by default. Private system deployment is strictly required before loading proprietary customer data.
