# HOCH RMF/ATO Cybersecurity Prompt Pack — Frequently Asked Questions

### Q1: Does Prompt Brain have authority to approve my system authorization?
**A**: No. Authorizing Officials (AO), ISSOs, and SCA auditors retain all regulatory approval authority. Prompt Brain acts as a compliance pre-screening co-pilot.

### Q2: Is my sensitive data leaked to external APIs?
**A**: No. The product is engineered to execute entirely locally against air-gapped instances of LM Studio and Ollama. All external provider pipelines are disabled in the default secure configuration.

### Q3: How are prompts validated?
**A**: Every prompt is subjected to a 9-dimensional dynamic scoring evaluation (completeness, specificity, verifiability, etc.) and tested against 30+ messy input cases.

### Q4: Can I run custom STIG checklists?
**A**: Yes. The system parses standard XML check structures and maps configuration fixes directly.
