# HOCH Prompt Brain — 30-Minute Demonstration Script

## 1. Opening Positioning (5 mins)
* **Opening**: "Thanks for joining. Today we are demonstrating how the HOCH Prompt Brain Factory accelerates RMF/ATO control narrative audits while ensuring complete local data containment."
* **Pain Discovery**: "How many hours does your team currently spend reviewing draft SSPs and checking for GPO failures?"

---

## 2. Walkthrough (18 mins)
* **SSP Narrative Critique**: Show a draft where administrator approvals are missing. Run the workflow and display the highlighted gap.
* **POA&M Resolution**: Triage a high vulnerability without target dates. Prompt Brain flags the omission immediately.
* **STIG Checklist Review**: Display a Windows Server GPO failure. Prompt Brain outputs the PowerShell command to remediate.
* **Human-in-the-Loop Clarification**: "Note that Prompt Brain flags missing parameters and asks for evidence instead of guessing or approving unconditionally. The final sign-off remains with you."
