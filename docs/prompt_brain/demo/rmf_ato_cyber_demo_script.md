This script walks through the live demonstration of the RMF/ATO Cybersecurity Prompt Pack RC1.

## 1. Introduction (2 mins)
* **Goal**: Frame the problem (lengthy ATO approval times, high audit preparation costs, messy compliance documentation).
* **Key Message**: Prompt Brain automates the first pass of security control narrative and compliance evidence audit review.
* **Disclaimer**: *Clearly state that this product does not replace ISSO, ISSM, SCA, or human cybersecurity judgment.*

---

## 2. Walkthrough Scenarios (10 mins)

### Scenario A: SSP Control Narrative Critique
* **Action**: Show an incomplete Access Control (AC-2) narrative.
* **Prompt Brain Execution**: Run the approved SOP prompt.
* **Key Visual**: Dashboard shows identified gaps (missing timeframe, undefined approvals) highlighted live.
* **Outcome**: A structured, correct list of remediation recommendations.

### Scenario B: Messy Nessus scan triage
* **Action**: Inject a scan output log with format anomalies.
* **Prompt Brain Execution**: Run Nessus-to-Risk prompt.
* **Outcome**: Dynamic trace outputs actionable mitigation steps mapping to specific NIST controls.
