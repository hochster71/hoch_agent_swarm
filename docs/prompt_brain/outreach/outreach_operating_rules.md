# HOCH Prompt Brain — Outreach Operating Rules

These operating rules must be strictly followed before any marketing, LinkedIn, or email sequence is triggered.

---

## 1. Compliance Core Rules
* **Human Approval Mandatory**: No outreach email or LinkedIn message may be sent automatically. All messages must be human-approved first.
* **No Sensitive Program Claims**: Do not mention customer programs, classified systems, or proprietary government data.
* **No Compliance Guarantees**: Under no circumstances should the text state or imply that Prompt Brain guarantees passing an SCA assessment or receiving an ATO.
* **No Authority-Replacement Language**: Do not imply that Prompt Brain replaces AOs, SCAs, or ISSOs. It is strictly a pre-screening copilot.
* **No Customer Data Collection**: Never request the recipient to send or upload their own sensitive compliance artifacts during outreach.
* **Sanitized Demo Data**: Walkthroughs and pilots must strictly run on the sanitized demo dataset.
