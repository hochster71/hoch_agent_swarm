# Page Map & Routing Blueprints

This page map outlines the information architecture and layout directions for each route in the HOCH AI OS Control Plane.

## Route Map

### 1. `/` — North Star Landing Page
- **Page Title**: HOCH AI OS — Shoot for the Moon
- **Layout Description**: Full-screen constellation map visualization. The 8 domain nodes (Personal, Family, Home, Business, Software Factory, Cybersecurity, Hobbies, Humanity) radiate from the central North Star metric container. Clicking a node opens an overview overlay or drills down into the sub-route.

### 2. `/control-plane` — Mission Control Overview
- **Page Title**: Mission Control Dashboard
- **Layout Description**: NASA/SpaceX-style firing room grid. Displays status widgets, telemetry streams, and core indicator cards for the active swarm process.

### 3. `/control-plane/assets` — Known Assets + Device Map
- **Page Title**: Device Service Registry
- **Layout Description**: Known asset registry network tree. Connects the MacBook Pro commander to the iMac model runtime, Dell, NEO, and mobile nodes.

### 4. `/control-plane/models` — Local Models + Model Router
- **Page Title**: Local Models Engine
- **Layout Description**: Dual-column model manager. Left: lists discovered Ollama and LM Studio model runtimes with their performance tiers. Right: displays active routing paths and lifecycle status (PASS, TRAINABLE, QUARANTINE).

### 5. `/control-plane/prompts` — Prompt Registry + Router
- **Page Title**: Prompt Control Plane
- **Layout Description**: Dynamic prompt inspector. Displays the 103 prompts in the library, current routing rules, and the prompt safety verification metrics.

### 6. `/control-plane/agents` — Agent Flight Deck
- **Page Title**: Agent Flight Deck
- **Layout Description**: Four-lane kanban style view showing running campaigns, active task logs, agent duty rosters, and current code-execution/signing waiver blocks.

### 7. `/control-plane/cyber` — Cyber Defense + ConMon
- **Page Title**: Cyber Defense Command
- **Layout Description**: High-contrast compliance dashboard tracking open findings, port exposures, DAST/SAST scan gates, and ConMon daily/weekly verification checklists.

### 8. `/control-plane/evidence` — Evidence Vault
- **Page Title**: Evidence Vault
- **Layout Description**: Linear chain-of-custody log displaying task validation manifests, SHA-256 integrity checkers, SBOM releases, and provenance bundles.

### 9. `/control-plane/approvals` — Human Approval Queue
- **Page Title**: Operator Decision Gate
- **Layout Description**: Lockscreen style interface staging blocked material actions with interactive Accept/Deny triggers.

### 10. `/factory` — Software Factory Overview
- **Page Title**: Hoch Application Software Factory
- **Layout Description**: Flow diagram of the software conveyor belt (Idea to Release package). Shows active project workspaces.

### 11. `/factory/apps` — Application Portfolio
- **Page Title**: Shipped App Portfolio
- **Layout Description**: Grid layout of compiled, signed, and released local applications.

### 12. `/factory/build-lanes` — Ephemeral Build Pipelines
- **Page Title**: Ephemeral Build Lanes
- **Layout Description**: Terminal-log style layout showing isolated dynamic dynamic workspaces during execution and automated cleanup.

### 13. `/factory/release` — Release Gate
- **Page Title**: Release Gatekeeper
- **Layout Description**: Compliance scoring table showing SBOM, signature checking, and final human sign-off status.

### 14. `/life` — Personal/Home/Family/Hobby OS
- **Page Title**: Life OS Constellation
- **Layout Description**: Dashboard mapping personal, family, home, and hobby goals and active missions.

### 15. `/life/home` — Home Operations
- **Page Title**: Home Operations Center
- **Layout Description**: Telemetry tracking home automation schedules, weather, backups, and stationary LAN nodes.

### 16. `/life/family` — Family Support
- **Page Title**: Family Support Portal
- **Layout Description**: Moonlight theme layout for scheduling, reminders, and child safety gates.

### 17. `/life/hobbies` — Hobby Lab
- **Page Title**: Hobby Experiment Lab
- **Layout Description**: Interactive sandbox tracking creative coding, game templates, and AI integrations.

### 18. `/life/humanity` — Positive Impact Missions
- **Page Title**: Humanity Projects
- **Layout Description**: Starfield board tracking open-source safety contributions and community-aligned tasks.
