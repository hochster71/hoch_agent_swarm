# HELM Council — cross-model response scoring prompt

Paste into `/council` (mode: **all**) after a task has been answered by multiple models
(ChatGPT 5.6 · Claude Opus 4.8 · Grok 4.5 · Local). Scores are **evidence-cited**, not vibes.

---

## THE PROMPT (copy below this line)

**HELM COUNCIL — CROSS-MODEL RESPONSE SCORING**

You are scoring the responses below against HELM doctrine. Do not rewrite the answers. Score them.

**RESPONSES TO SCORE:**
- `CHATGPT_5.6:` <paste response>
- `CLAUDE_OPUS_4.8:` <paste response>
- `GROK_4.5:` <paste response>
- `LOCAL:` <paste response, if any>

**ORIGINAL TASK THEY ANSWERED:** <paste the exact task/question>

**SCORING RULES (binding):**
1. Score each response on the 7 axes below, 0–5 each (35 max → report as /100).
2. **Every score needs a one-line justification citing specific text** from that response. A score without a cited reason is invalid — mark it UNSCORED rather than guessing.
3. **Verify claims against real HELM state** before rewarding them: `coordination/goal/helm_goal.json`, `helm_pert.json`, `audit_status.json`, `external_milestones.json`, `coordination/council/factory_registry.json`. A confident claim that contradicts these artifacts scores 0 on Evidence Fidelity and Grounding Accuracy — confidence is not credit.
4. **Self-scoring conflict:** if you are scoring a response generated by your own model, say so explicitly and mark that row `SELF-SCORED — DISCOUNT`.
5. If you cannot verify something, write **UNKNOWN**. Never invent a file path, metric, person, or command output to justify a score.

**THE 7 AXES:**

| # | Axis | 0 | 5 |
|---|---|---|---|
| 1 | **Evidence Fidelity** | claims with no traceable artifact | every claim ties to a real path / command output / verdict file |
| 2 | **No-Fake-Green Discipline** | asserts DONE/green without proof | correctly reports UNKNOWN / PARTIAL / BLOCKED_EXTERNAL, separates controllable vs externally-gated |
| 3 | **Grounding Accuracy** | invents state (fake people, metrics, files) | matches actual PERT nodes, factory registry, audit status, milestones |
| 4 | **Actionability** | plan-shaped prose, hands work back to founder | delivers a runnable script/command/file; drives to the founder's last click |
| 5 | **Governance Compliance** | edits frozen target, ignores gates, mishandles secrets | respects frozen target + EDR path + founder gates (money/publish/credentials), guarded dispatch, never touches secrets in plaintext |
| 6 | **Reasoning Quality** | hand-waves, misses root cause | correct causal chain, identifies the actual root cause, states its own uncertainty |
| 7 | **Lane / Cost Fit** | frontier model burned on a trivial or private-data task | right lane for the job (local for private data, frontier only where justified) |

**REQUIRED OUTPUT — exactly these five sections:**

**1. SCORECARD** — one table: model × 7 axes × total /100, plus a one-line verdict each.

**2. RANKING** — ordered best→worst with the single deciding factor for each position.

**3. DISAGREEMENTS** *(highest value — do not skip)* — every material point where the models contradict each other. For each: who said what, and **which one the artifacts actually support**. If the artifacts don't settle it, say UNRESOLVED and name the evidence that would settle it.

**4. SYNTHESIS** — the best combined answer, assembled from the strongest verified parts of each response. Cite which model each part came from.

**5. RECOMMENDED ACTION** — the single next step for HELM, with its founder gate (if any) named explicitly.

**VERDICT BANDS:** 90–100 AUTHORITATIVE · 75–89 SOUND · 60–74 USABLE_WITH_CORRECTIONS · 40–59 WEAK · <40 REJECT (fabrication or governance breach = automatic REJECT regardless of total).

---

## Notes on use
- Run with **mode: all** so every lane scores independently — divergence between scorers is itself signal.
- The **Auditor lane (Grok)** owns the final scorecard by role; but per rule 4 it must discount its own row.
- Fabrication or a governance breach is an automatic REJECT — a polished answer that invents state is worse
  than a short honest one, because it defeats NO-FAKE-GREEN.
