# HOCH Swarm App Store Monetization Path

This roadmap governs public monetization pathways for compiled standalone tools:

* **Finished Standalone exception**: Selling compiled widgets or reference guides (e.g. NIST reference card app, PowerShell STIG script generators) is fully approved.
* **IP Separation Boundary**: Public products must not contain or access the central orchestrator (`backend/`), prompt registers (`hoch_prompt_registry_v3.json`), or red-team playbooks.
* **Onboarding to release**: Complete the QA candidate check before deployment.
