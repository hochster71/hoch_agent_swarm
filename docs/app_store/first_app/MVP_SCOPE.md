# MVP Scope — RMF Evidence Review Companion

This document lists the concrete features included in the initial Minimum Viable Product (MVP) release:

## In-Scope Features
* **NIST Checklist Viewer**: Searchable checklist mapping NIST SP 800-53 Rev 5 control groups.
* **Status Flags**: Users can mark controls as "Compliant", "Non-Compliant", or "Not Applicable".
* **Local Export**: Export checklist states to a local CSV/PDF report file.
* **On-device Storage**: Local SQLite DB storing checklist configurations.
