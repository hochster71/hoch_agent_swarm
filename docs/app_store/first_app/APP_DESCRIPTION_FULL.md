RMF Compliance Review Companion is a fully offline, privacy-first tool designed for cybersecurity consultants, ISSOs, and ISSMs mapping NIST SP 800-53 security controls.

Key Features:
* Detailed control checklists coveringRev 5 baselines.
* Set status flags locally.
* Generate clean CSV reports on device.
* Purely offline: No telemetry, no account registration, no remote server APIs.
