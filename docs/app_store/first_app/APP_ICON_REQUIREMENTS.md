# App Icon Asset Requirements

Dimensions for App Store package validation:

* **App Store Icon**: `1024 x 1024` pixels (PNG, no alpha channel)
* **iPhone Icon (2x)**: `120 x 120` pixels
* **iPhone Icon (3x)**: `180 x 180` pixels
* **iPad Icon (2x)**: `152 x 152` pixels
* **macOS Icon**: `512 x 512` pixels
