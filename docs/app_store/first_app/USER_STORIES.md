# User Stories — RMF Evidence Review Companion

User validation scenarios for the compliance companion app:

## Story 1: Selecting Control Group
* **As an** ISSO
* **I want to** select specific NIST 800-53 control families (e.g., AC, IA)
* **So that** I can review evidence validation checklists for those families.

## Story 2: Exporting Compliance Report
* **As a** compliance consultant
* **I want to** export progress to a local file
* **So that** I can present it in review meetings.
