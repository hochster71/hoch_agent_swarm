# HOCH Swarm App Release Checklist

Perform these 10 steps before submitting candidates to public stores:

1. **Confirm Standalone Execution**: Run without external backend connection.
2. **Scan for Private IP**: Verify prompt templates are compiled out.
3. **Verify Sandbox**: Confirm no system logs are leaked.
4. **App Store Metadata**: Supply descriptions without claiming ATO authority.
5. **Privacy Policy**: Attach standard policy.
6. **Terms of Service**: Attach Terms.
7. **Support Route**: Connect the feedback form.
8. **Monetization Check**: Verify payment gateways are operational.
9. **Objection Pre-screen**: Inspect for potential store rejection reasons.
10. **Michael Sign-off**: Obtain explicit written approval before submitting.
