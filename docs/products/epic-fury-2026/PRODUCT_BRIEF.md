# Product Brief: Epic Fury 2026

* **Name**: Epic Fury 2026
* **Status**: `TARGET_SELECTED`
* **Vetting Pipeline Phase**: `Phase 2`

---

## 1. Product Description
Epic Fury 2026 is a dark-mode, military HUD-style tactical intelligence dashboard. It serves as a visual control interface displaying real-time AI agent status feeds, tactical updates, and DMO canvas simulations.

## 2. Target User
AI researchers, drone/simulation operators, and mission planners looking for an integrated tactical command center UI that visually correlates AI agents' reasoning logs with simulated geographic intelligence.

## 3. Core Value Proposition
* **Real-time Tactical Feed**: Supabase real-time pub/sub feed displaying incoming intel reports with confidence metrics.
* **AI Agent Roster Grid**: Real-time status cards of active agent nodes showing live logs, telemetry, and metrics.
* **DMO Canvas Simulation**: Interactive HTML5 canvas map representing DMO (Hormuz Strait) simulation scenarios.

## 4. Technical Specifications
* **Framework**: Next.js 15 (App Router, strict TypeScript)
* **Monetization**: Paid access subscription path with Stripe and RevenueCat.
* **Hybrid Wrapper**: Capacitor CLI & Capacitor iOS wrapper.
