# HOCH AI OS Visual Control Plane — Phase V1 Static Mockups

This directory contains the dark-theme, static HTML/CSS mockups designed for the HOCH AI OS.

## Paradigm Structure

### 1. Landing: North Star Life OS
- **Entry point**: `index.html`
- **Purpose**: Maps personal, family, home, business, and humanity goals into a deep-space constellation radiating from the weekly outcomes score box.

### 2. Operations: Mission Control Center
- **Entry point**: `control-plane.html`
- **Purpose**: Displays the real-time operational dashboard including assets, runtimes, prompt registries, approvals, and ConMon status.

### 3. Production: Hoch Application Software Factory
- **Entry point**: `factory.html`
- **Purpose**: Implements the pipeline conveyor lane (Idea-to-ConMon) for ephemeral application workspaces.

---

## Technical Details

- **Visual Tokens**: Managed entirely in `styles.css` using CSS custom properties.
- **Theme**: Strict dark mode (#030303 background, no light theme).
- **Accessibility**: All indicators pair colors with text/icons; keyboard focus states are enabled; prefers-reduced-motion media queries disable loops/pulses.
- **Data Status**: No fake live telemetry; unverified or sample data points are explicitly marked as `SAMPLE` or `SIMULATED`.
